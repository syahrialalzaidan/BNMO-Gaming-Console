## Deskripsi 
5 orang mahasiswa sti disuruh ngurusin BNMO (dibaca: Binomo). BNMO (dibaca: Binomo) adalah sebuah robot video game console yang dimiliki oleh Indra dan Doni. Dua bulan yang lalu, ia mengalami kerusakan dan telah berhasil diperbaiki. Sayangnya, setelah diperbaiki ia justru mendapatkan lebih banyak bug dalam sistemnya. Oleh karena itu, Indra dan Doni menyuruh kita (dengan asumsi kita programmer handal) untuk ulang memprogram robot video game console kesayangannya.
